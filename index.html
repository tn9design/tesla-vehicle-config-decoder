Tesla Vehicle Config Decoder (Unofficial)

Build a Tesla image URL that matches your vehicle config. Uses Tesla's public compositor endpoint. No login. No guarantees.

Notes:
  • Use "2021-2023 legacy visuals" for old `v1/compositor` angles like `STUD_3QTR_V2`.
  • Use "Extra option codes" for packages not shown in dropdowns (for example: `$APF2,$APBS`).
  • Accessories (Home Charger, liners, etc.) do not affect the render URL.
  • Wheels and interiors can be trim-scoped (options change by trim).
  • Modern Model S steering only affects INTERIOR view; legacy Model S steering can affect exterior renders too.

If the preview does not load, Tesla may be rate-limiting or the endpoint may have changed.
